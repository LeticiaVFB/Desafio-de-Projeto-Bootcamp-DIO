Comandos básicos no Git

Esse é somente um guia rápido de comandos do Git para o Desafio de Projeto do Bootcamp Orange Tech + da DIO.

O Git funciona como um programa de controle de versões distribuído. Para usá-los precisamos de alguns comandos. Veremos a seguir os básicos:

1. Git Init - É o começa para iniciar uma nova configuração a um repositório.
2. Git Add - Adiciona uma alteração ao repositório. E adiciona ele para o status de staged.
3. Git Commit - Cria um commit, ou seja, ele salva o estado atual do seu repositório.
4. Git Push - "Push" significa "empurrar". Ele vai empurrar o seu repositório para algum lugar, por exemplo: O GitHub.
5. Git Pull - Faz o oposto do Push. Pull significa "puxar". Então ele puxa o repositório para o seu computador.
6. Git Status - Com esse comando você consegue saber qual o status do seu repositório.

Existem muitos outros comandos, mas esses são os mais básicos e os mais usados com frequência.
